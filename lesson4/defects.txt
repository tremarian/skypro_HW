Функция contains, принимает пустую строку для обработки.

Указать один из следующих наборов параметров
    ('', '',). Полученный результат будет True
    ('', '123'). Полученный результат будет False
При этих данных функция отрабатывает корректно. Однако, вероятно, будет лучше запретить ввод пустой строки, т.к. это не имеет смысла.
